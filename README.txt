Grocery Store Management Project

This project implements a Grocery Store Management system using Flask and SQLAlchemy.

Table of Contents:
- Prerequisites
- Installation
- Usage

Prerequisites:
- Python 3.x
- Flask
- Flask-SQLAlchemy

Installation:
1. Install the required dependencies:
   pip install -r requirements.txt

Usage:
1. Start the Flask development server:
   python app.py

2. Open a web browser and go to http://localhost:5000 to access the application.

3. Use the following credentials to log in and explore the functionalities:

   Admin Login:
   Username: admin
   Password: password

   User Login:
   You can create a new user account through the registration page and log in using the provided credentials.
